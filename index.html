PUAE by GnoStiC

Amiga Emulator (for Pandora & Unixish systems)

Dependencies

pandora and/or a unixish system

Install

step1) ./bootstrap.sh default step2) ./configure optimized step2) ./configure --enable-gccopt step3) make
